# SOAE-Dashboard

A lightweight compliance tracker that runs entirely from a flash drive. The dashboard stores its data in browser local storage, so you can open the HTML file directly without a server.

## Getting started

1. Copy the repository folder to your flash drive.
2. Open `index.html` in a modern browser (Chrome, Edge, Firefox, or Safari).
3. Add compliance items and use the filter to focus on overdue or completed reports.

> Tip: If you move to another computer, copy the full folder so the HTML, CSS, and JavaScript stay together. Your entries are stored in the browser you used, so export by copying the data in that browser if needed.

## Development

Install dependencies and run checks:

```bash
npm install
npm run lint
npm test
```
