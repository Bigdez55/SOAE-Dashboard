## Makefile Skills (15)

### SK-085: Two-PCH Architecture (NOSSE vs SSE)
PCH_NOSSE for kernel core (~75 files, -mno-sse). PCH_SSE for XMIND/XDISP (~48 files, -msse -msse2). AVX2/AESNI files excluded from PCH (feature superset conflict).
- **File:** `Makefile:99-118`

### SK-086: Per-File CFLAGS Overrides
CFLAGS_XMIND (-msse -msse2), CFLAGS_XMIND_AVX2 (-mavx2 -mfma), CFLAGS_AESNI (-maes -mpclmul -msse4.1). Explicit Makefile rules per subsystem.
- **File:** `Makefile:74-84`

### SK-087: Ccache Integration
`CC = ccache clang-18`. Transparent caching. Unity builds + ccache = cold 120s → warm 8s.
- **File:** `Makefile:21`

### SK-088: .DEFAULT_GOAL Declaration
`.DEFAULT_GOAL := all` — prevents PCH rule (first target) from becoming default goal.
- **File:** `Makefile:96`

### SK-089: Order-Only Prerequisites for PCH
`$(BUILD)/kernel/core/%.o: core/%.c | $(PCH_NOSSE)` — PCH must exist but doesn't trigger rebuild if older.
- **File:** `Makefile:590`

### SK-090: Pattern Substitution for Output Dirs
`KERNEL_OBJS = $(patsubst %.c,$(BUILD)/kernel/%.o,$(KERNEL_SRCS))` — maps source tree to flat build tree.
- **File:** `Makefile:446-529`

### SK-091: -pipe Compiler Flag
Pipes between compiler stages instead of temp files. ~5% faster on large builds.
- **File:** `Makefile:36`

### SK-092: -fvisibility=hidden
All symbols default hidden. Only explicitly exported symbols are visible. Reduces symbol table bloat.
- **File:** `Makefile:36`

### SK-093: Intentional -fno-stack-protector + Manual Canaries
Disable GCC stack protector (requires libc __stack_chk_fail). Replace with RDRAND-seeded manual canary.
- **File:** `Makefile:29-33`

### SK-094: Boot-Kernel Size Reduction via -D
`-DXBLOB_INDEX_SLOTS=4096u -DTCP_TCB_MAX=16u -DXFRAME_FB_MAX_W=1280u` etc. Production uses full sizes.
- **File:** `Makefile:32-40`

### SK-095: #ifndef Guards for -D Compatibility
Every overridable `#define` wrapped in `#ifndef`. Zero redefinition warnings.
- **File:** Pattern across all headers

### SK-096: compile-check vs link Target Separation
`make compile-check` validates syntax without linking. `make all` does full link. CI uses compile-check for fast feedback.
- **File:** `Makefile:557-570`
