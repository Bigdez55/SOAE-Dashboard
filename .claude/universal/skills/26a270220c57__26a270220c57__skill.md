# Go Microservices Build Verification

Build all three Go services and run unit tests.

---

## Step 1 — market-gateway

```bash
cd backend/go/market-gateway

echo "=== market-gateway ==="
go build ./... 2>&1
if [ $? -eq 0 ]; then
  echo "BUILD: PASS"
else
  echo "BUILD: FAIL"
fi

go vet ./... 2>&1 | head -10
echo "VET: done"

go test ./... -timeout 30s 2>&1 | tail -10
```

## Step 5 — OCC Regex Verification (order-router)

```bash
# Verify the options symbol regex is correct
cd backend/go/order-router
grep -n "isOptionsSymbol\|OCC\|regexp" internal/execution/*.go 2>/dev/null | head -10
```

Expected: regex pattern `^[A-Z]{1,6}\d{6}[CP]\d{8}$`
