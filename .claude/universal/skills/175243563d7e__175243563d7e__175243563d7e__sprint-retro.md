# GEN.OS Sprint Retrospective Protocol

Run after sprint close commit. Output saved to agent memory.

## Retrospective Template

```
SPRINT N RETROSPECTIVE
Date: [YYYY-MM-DD]
Commit: [hash]
Participants (agents): [list]

── DELIVERY ──────────────────────────────────────────────────────────
Planned deliverables: N
Delivered: N
Compile check count: N/N (target met / missed by N)
Test count: N/N
Lines of code: N (N new files)

── WHAT WORKED ───────────────────────────────────────────────────────
1. [Technical win]
2. [Process win]
3. [Agent collaboration win]

── WHAT DIDN'T ───────────────────────────────────────────────────────
1. [Technical blocker and resolution]
2. [Process friction]
3. [Agent gap or conflict]

── TECHNICAL DEBT INCURRED ───────────────────────────────────────────
| Item | File | Impact | Target Sprint |
|------|------|--------|---------------|
| [debt item] | [file] | [P1/P2/P3] | Sprint N+M |

── ARCHITECTURAL DECISIONS MADE ─────────────────────────────────────
| ADR | Decision | Rationale |
|-----|----------|-----------|
| ADR-NNN | [decision] | [why] |

── SPRINT N+1 INPUTS ────────────────────────────────────────────────
Dependencies for next sprint:
- [Module A API] — stable as of [file:line]
- [Header B] — finalized, no changes expected

Risk items:
- [Risk] — mitigation: [plan]

── AGENT PERFORMANCE ────────────────────────────────────────────────
| Agent | Deliverable | Quality | Notes |
|-------|-------------|---------|-------|
| [agent] | [task] | [G/A/R] | [note] |

── ACTION ITEMS ─────────────────────────────────────────────────────
| # | Action | Owner Agent | Due |
|---|--------|-------------|-----|
| 1 | [action] | [agent] | Sprint N+1 |
```
