# From-Scratch Subsystem Engineering — 45 Skills

Build display, storage, network, AI, and sync subsystems with zero third-party dependencies.

## Display Stack (6 skills)

### SK-121: Raw DRM/KMS Mode Setting
Zero-dep DRM: ioctl numbers hand-defined. Open /dev/dri/card0, enumerate connectors, parse EDID, select preferred mode, create dumb buffer, mmap for scanout.
- **File:** `devices/desktop/display/xdisp/drm.c`

### SK-122: Scene Graph Compositor
Z-sorted doubly-linked list of surfaces. Double-buffered back/front swap. Damage tracking avoids full recomposite. GOP fallback when DRM unavailable.
- **File:** `devices/desktop/display/xcomp/compositor.c`

### SK-123: Porter-Duff SRC_OVER Integer Blend
`BLEND_MUL255(a,b) = (a*b + (a*b>>8) + 1) >> 8` — exact for all inputs. Three fast paths: skip transparent, direct copy opaque, full blend.
- **File:** `devices/desktop/display/xcomp/blend.c:297-318`

### SK-124: 2D Vector Graphics Engine
De Casteljau flattening (0.25px tolerance). Non-zero winding scanline fill with 4x vertical supersampling. Miter/bevel/round joins.
- **File:** `devices/desktop/display/xcomp/vector.c`

### SK-125: OpenType Font Rasterizer
sfnt parser, cmap Format 4, quadratic B-spline glyph outlines, scanline rasterizer with 4x AA, LRU glyph cache, 4MB font arena.
- **File:** `devices/desktop/display/xcomp/fonts.c`

### SK-126: GOP Direct Blit Pipeline
XFRAME paint → g_sw_fb → XCOMP blend → GOP framebuffer via HHDM physical address. Bypasses DRM when unavailable.
- **File:** `devices/desktop/ui/xframe/runtime/main.c:108`

## Storage Stack (5 skills)

### SK-135: B+ Tree KV Store
Order-256 leaf, order-128 internal. 4096-byte pages. Variable-length records with tail-growing offset table. Leaf chain via next_leaf.
- **File:** `store/xstore/btree/btree.c`

### SK-136: Write-Ahead Log (REDO-only)
Ring buffer with CRC32C per entry. Auto-checkpoint at threshold. Crash recovery: collect COMMIT txn_ids, replay their ops, discard partial.
- **File:** `store/xstore/wal/wal.c`

### SK-137: Content-Addressable Blob Store (XBLOB)
SHA-256 key = hash of content. 1M-slot open-addressing hash table (linear probing, tombstone deletion, 75% load factor). GC via saturating refcount.
- **File:** `store/xblob/include/xblob.h`

### SK-138: I/O Callback Abstraction
Function pointer structs (read, write, sync) abstract NVMe, RAM disk, or test buffer backends. Zero coupling to hardware.
- **File:** `xstore.h`, `xblob.h`

### SK-139: CRC32C Integrity on All On-Disk Structures
Castagnoli CRC. Checksum field zeroed before compute. Page bytes [8, PAGE_SIZE).
- **File:** `xstore.h`, `wal.c`, `xblob.h`

## Network Stack (6 skills)

### SK-140: TCP State Machine (RFC 793)

### SK-141: SWIM Gossip Protocol (256 nodes)
1s probe interval, k=3 indirect probers, 8 piggybacked updates per message. O(log N) convergence. UDP-based.

### SK-143: Ring Buffer (TCP)

### SK-144: Byte Order Helpers

## AI Inference (9 skills)

### SK-149: SIMD Runtime Dispatch

### SK-151: RoPE Precomputation

### SK-152: KV Cache Management

### SK-153: Stable Softmax

- **File:** `tensor.c`, `transformer.c:126-152`

### SK-154: AVX2+FMA3 Dot Product + Q4_0 Dequant
8-wide YMM VFMADD231PS loop + horizontal reduction. Q4_0 nibble unpack → float scale → AVX2 dot. ~8x throughput.
