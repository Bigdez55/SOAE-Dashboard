# GEN.OS API Contract Specification

For every IPC/D-Bus interface or REST endpoint that crosses a service boundary.
File: `docs/contracts/[interface-name].md`

## IPC / D-Bus Interface Contract

```markdown
# [Interface Name] — IPC Contract

**Interface path:** `genos.[domain].[Interface]`
**Transport:** D-Bus / UNIX socket / shared memory ring
**Introduced:** Sprint N
**Status:** Draft | Stable | Deprecated

## Error Codes

| Code | Name | Meaning |
|------|------|---------|
| 0 | OK | Success |
| 1 | ERR_PERM | Insufficient XKABI rights |
| 2 | ERR_NOT_FOUND | Resource does not exist |
| 3 | ERR_BUSY | Resource locked |
| 255 | ERR_INTERNAL | Internal service error |

## REST Endpoint Contract (FastAPI services)

```markdown
# [Service Name] REST API

**Base URL:** `http://[service]:8000/api/v1`
**Auth:** Bearer JWT (all endpoints except /health)

## GET /[resource]

**Purpose:** [Description]
**XKABI right required:** [right or "N/A — user-space only"]

**Response 200:**
```json
{
  "id": "string",
  "field": "value",
  "created_at": "ISO8601"
}
```

**Response 401:** JWT missing or invalid
**Response 403:** Insufficient rights
**Response 404:** Resource not found
**Response 503:** Service unhealthy (never returns "degraded")

**Latency SLA:** P50 < [N]ms, P99 < [N]ms
```
