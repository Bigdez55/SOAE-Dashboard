# GEN.OS Compliance Check — Full Platform Validation

Run before sprint close and before every PR to `main`.

## Gate 0 — Language Policy

```bash
python platform/check_language_policy.py
```
Exit 0 = PASS.

## Gate 1 — Kernel Compile (N/N)

Run `/compile-check` skill. Record pass count. Must meet sprint target.

## Gate 2 — Static Analysis (all 4)

Run `/static-analysis` skill. All tools must exit 0.

## Gate 3 — Tests

```bash
make -f Makefile.rfc002 run-tests           # 80/80
pytest -q                                    # all pass
python -m pytest xisc/conformance/ -q       # 26/26 (Linux)
./build/rfc002/test_xkabi_barriers          # 14/14
./build/rfc002/xkabi_conformance            # 26/26
```

## Gate 4 — Security

```bash
trivy fs . --severity CRITICAL,HIGH --exit-code 1 \
  --skip-dirs .claude,node_modules,build/rootfs
pip-audit --desc --fix=no
npm audit --audit-level=high
```

## Gate 5 — Memory Policy (Kernel)

```bash
# 0 dynamic allocations in freestanding code
grep -rn "malloc\|calloc\|realloc\|free(" \
  kernel/ display/ ui/xframe/ --include="*.c" --include="*.h" \
  | grep -v "^.*//.*malloc"
```
Expected: 0 matches.

## Gate 6 — XKABI Rights Policy

```bash
# No raw hex rights in .gsd files
grep -rn "Capabilities.*0x" init/gensd/services/ 2>/dev/null
```
Expected: 0 matches.
