# Stop vLLM VM — Cost Saver (~$1.50/hr)

Gracefully stop the L4 GPU VM after trading hours.

---

## Step 1 — Confirm Bot is Stopped First

```bash
# Do NOT stop vLLM while bot is actively trading
curl -s https://api.elsontrade.com/api/v1/auto-trading/status \
  -H "Authorization: Bearer $TOKEN" 2>/dev/null | python3 -c "
import sys, json
try:
  d = json.load(sys.stdin)
  print('Bot running:', d.get('is_running', 'unknown'))
except: print('Could not determine bot status')
"
```

If bot is RUNNING → stop it first via `/stop-bot`, then return here.

## Step 2 — Check Current VM State

```bash
gcloud compute instances describe elson-dvora-training-l4-2 \
  --zone=us-west1-a \
  --project=elson-33a95 \
  --format="value(status)"
```

If already `TERMINATED` → already stopped, nothing to do.

## Step 3 — Stop the VM

```bash
gcloud compute instances stop elson-dvora-training-l4-2 \
  --zone=us-west1-a \
  --project=elson-33a95
```

## Step 4 — Confirm Stopped

```bash
sleep 15
gcloud compute instances describe elson-dvora-training-l4-2 \
  --zone=us-west1-a \
  --project=elson-33a95 \
  --format="value(status)"
```

Expected: `TERMINATED`
