# Start vLLM VM — elson-dvora-training-l4-2

Start the L4 GPU VM for AI inference. Allow 3-5 minutes for full startup.

---

## Step 1 — Check Current State

```bash
gcloud compute instances describe elson-dvora-training-l4-2 \
  --zone=us-west1-a \
  --project=elson-33a95 \
  --format="value(status)"
```

If already `RUNNING` → nothing to do, report status and exit.

## Step 2 — Start the VM

```bash
gcloud compute instances start elson-dvora-training-l4-2 \
  --zone=us-west1-a \
  --project=elson-33a95
```

## Step 3 — Wait for Boot (poll every 30s, max 5 min)

```bash
for i in $(seq 1 10); do
  STATUS=$(gcloud compute instances describe elson-dvora-training-l4-2 \
    --zone=us-west1-a --project=elson-33a95 --format="value(status)")
  echo "Attempt $i/10: VM status = $STATUS"
  if [ "$STATUS" = "RUNNING" ]; then break; fi
  sleep 30
done
```

## Step 4 — Verify vLLM Reachable via API

```bash
sleep 60  # Allow vLLM server to fully initialize
curl -s https://api.elsontrade.com/health | python3 -c "
import sys, json
data = json.load(sys.stdin)
print('Health:', data.get('status'))
print('vLLM:', data.get('vllm_status', 'not reported'))
"
```
