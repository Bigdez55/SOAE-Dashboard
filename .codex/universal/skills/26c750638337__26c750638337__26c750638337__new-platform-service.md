# GEN.OS New Platform Service — Python FastAPI Template

Creates a production-ready microservice in `platform/services/[service]/`.

## Directory Structure

```
platform/services/[service]/
├── main.py           # FastAPI app entry point
├── models.py         # SQLAlchemy models
├── routes.py         # API route handlers
├── service.py        # Business logic layer
├── config.py         # Settings (pydantic BaseSettings)
├── requirements.txt  # Pinned dependencies
├── Dockerfile        # Multi-stage build
└── tests/
    ├── test_routes.py
    ├── test_service.py
    └── conftest.py
```

## `main.py`

```python
"""
platform/services/[service]/main.py — [Service Name] FastAPI entrypoint.

Copyright (c) 2026 GEN.OS Project. All rights reserved.
SPDX-License-Identifier: LicenseRef-Proprietary

Task: S[N]-SVC-[ID]
"""
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware
from contextlib import asynccontextmanager
from .routes import router
from .config import settings
import logging

logging.basicConfig(level=logging.INFO)
logger = logging.getLogger(__name__)


@asynccontextmanager
async def lifespan(app: FastAPI):
    logger.info("[service] starting up")
    # TODO: init DB, cache connections
    yield
    logger.info("[service] shutting down")


app = FastAPI(
    title="[Service Name]",
    version="0.1.0",
    lifespan=lifespan,
)

app.add_middleware(
    CORSMiddleware,
    allow_origins=settings.allowed_origins,
    allow_methods=["GET", "POST", "PUT", "DELETE"],
    allow_headers=["Authorization", "Content-Type"],
)

app.include_router(router, prefix="/api/v1")


@app.get("/health")
async def health() -> dict:
    """Health probe — k3s liveness + readiness gate."""
    return {"status": "healthy", "service": "[service]"}
```

## `config.py`

```python
from pydantic_settings import BaseSettings


class Settings(BaseSettings):
    database_url: str = "sqlite:///./[service].db"
    allowed_origins: list[str] = ["http://localhost:3000"]
    jwt_secret: str  # Must be set via env — no default
    debug: bool = False

    class Config:
        env_file = ".env"
        env_prefix = "[SERVICE]_"


settings = Settings()
```

## Health Endpoint Contract

The `/health` endpoint MUST:
- Return `{"status": "healthy"}` when operational
- Return HTTP 503 (NOT 200 with `"degraded"`) when unhealthy
- Respond in < 50ms under normal load

k3s manifest liveness probe:
```yaml
livenessProbe:
  httpGet:
    path: /health
    port: 8000
  initialDelaySeconds: 5
  periodSeconds: 10
  failureThreshold: 3
```

## Constraints

- [ ] Language: Python only (no Go, no Rust)
- [ ] No secrets in source — all from env vars
- [ ] JWT validation on all non-health endpoints
- [ ] Health returns `"healthy"` string — never `"degraded"`
- [ ] Database connections use retry loop (tenacity or manual)
- [ ] `requirements.txt` uses pinned versions (`==`, never `>=`)
- [ ] `bandit -r platform/services/[service]/ -ll` passes
- [ ] `ruff check platform/services/[service]/` passes
- [ ] `pytest platform/services/[service]/tests/ -q` passes
