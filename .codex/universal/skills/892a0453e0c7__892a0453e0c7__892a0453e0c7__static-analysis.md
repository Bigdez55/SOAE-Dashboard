# GEN.OS Static Analysis — All 4 Tools

Run before every PR. Zero tolerance: 0 warnings, 0 violations, 0 CVEs.

## 1 — cppcheck (C)

```bash
cppcheck --enable=warning,performance,portability \
  --error-exitcode=1 \
  --suppress=missingInclude \
  --suppress=missingIncludeSystem \
  -I pal/include -I kernel/xenos/include -D__x86_64__ \
  pal/src/pal_aether.c \
  kernel/xenos/core/main.c kernel/xenos/mm/pmm.c kernel/xenos/mm/vmm.c \
  kernel/xenos/mm/heap.c kernel/xenos/sched/contra_rotation.c \
  kernel/xenos/cap/xkabi_capabilities.c \
  kernel/xenos/drv/intel_igpu.c kernel/xenos/drv/nvme.c \
  init/gensd/main.c init/gensd/service.c init/gensd/supervisor.c \
  init/gensd/journal.c init/gensd/splash.c \
  display/xdisp/*.c display/xcomp/*.c \
  ui/xframe/runtime/*.c ui/xframe/widgets/*.c ui/xframe/sync/*.c
echo "cppcheck exit: $?"
```

## 2 — Ruff (Python)

```bash
ruff check .
ruff format --check .
echo "ruff exit: $?"
```

## 3 — ESLint (TypeScript)

```bash
cd compositor/genos-shell
npx eslint 'src/**/*.{ts,tsx}' 'electron/**/*.ts' --max-warnings 0
echo "ESLint exit: $?"
cd -
```

## 4 — Bandit + Trivy + pip-audit (Security)

```bash
# SAST — HIGH+CRITICAL only
bandit -r platform/ ai/ -ll \
  --exclude platform/services/identity/tests,platform/services/sync/tests
echo "bandit exit: $?"

# CVE scan — filesystem
trivy fs . --severity CRITICAL,HIGH --exit-code 1 --ignore-unfixed \
  --skip-dirs .claude,node_modules,build/rootfs
echo "trivy exit: $?"

# Python dependency CVEs
pip-audit --desc --fix=no
echo "pip-audit exit: $?"

# Node dependency CVEs
npm audit --audit-level=high
echo "npm-audit exit: $?"
```

## Pass Criteria

| Tool       | Required result         |
|------------|------------------------|
| cppcheck   | Exit 0, 0 warnings     |
| ruff       | Exit 0, 0 violations   |
| ESLint     | Exit 0, 0 warnings     |
| bandit     | 0 HIGH or CRITICAL     |
| trivy      | 0 CRITICAL or HIGH CVE |
| pip-audit  | 0 vulnerabilities      |
| npm audit  | 0 high vulnerabilities |
