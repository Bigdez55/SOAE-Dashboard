# Market Pulse — Event Impact Analysis

Analyze a market event and trace its causal impact on the portfolio.

---

## Step 1 — Identify the Event

Parse the user's input to extract:
- **Event type:** Fed decision / earnings / macro data / geopolitical / sector news
- **Event:** e.g., "Fed raised rates 25bps", "NVDA beat earnings by 30%", "Taiwan earthquake"
- **Magnitude:** Quantify if possible (bps, %, EPS beat/miss)

## Step 2 — Current Portfolio Exposure

```bash
TOKEN=$(curl -s -X POST https://api.elsontrade.com/api/v1/auth/login \

```

## Step 3 — Causal Impact Analysis

Trace event → sector → symbol impact chain:

**For Fed rate decisions:**
- Rate ↑ → Bond yields ↑ → Growth stocks ↓ (tech/QQQ), Financials ↑ (XLF)
- Rate ↓ → Opposite; REIT/Utilities ↑

**For earnings surprises:**
- Beat + raised guidance → stock ↑ 5-15%, sector sympathy ↑ 1-3%
- Miss + lowered guidance → stock ↓ 10-20%, sector contagion

**For geopolitical events:**
- Supply chain disruption → semiconductor shortage → NVDA/AMD/TSMC exposed
- Energy shock → oil producers ↑ (XOM, CVX), consumers ↓ (airlines)

**For macro data (CPI, NFP, GDP):**
- Hot CPI → hawkish Fed fear → growth ↓, commodities mixed
- Weak NFP → recession fear → defensive sectors ↑ (XLU, XLP)
