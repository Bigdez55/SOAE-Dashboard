# Sprint 3 — XSEC Security Module Design & Scaffold

XSEC is the cryptographic and attestation layer for GEN.OS.
Depends on: TPM (aetherboot/src/tpm.c), XKABI capability system.

## Architecture

```
security/xsec/
├── include/
│   ├── xsec.h          # Public API
│   └── xsec_types.h    # Key types, cert types, attestation structs
├── xsec_core.c         # Init, key registry, policy enforcement
├── xsec_tpm.c          # TPM 2.0 interface (extends aetherboot/src/tpm.c)
├── xsec_crypto.c       # SHA-256, AES-256-GCM (hardware-accelerated via AES-NI)
├── xsec_attest.c       # Remote attestation quote generation
├── xsec_cert.c         # X.509 certificate parsing (DER, no libc)
└── xsec_policy.c       # Security policy evaluation engine
```

## Design Constraints

| Constraint | Specification |
|------------|--------------|
| Crypto | AES-NI instructions only — no software fallback for production |
| Key storage | TPM 2.0 sealed storage (HP EliteBook TPM) |
| Memory | Static key registry (64 slots). No heap. |
| XKABI right | `XK_RIGHT_CRYPTO_KEY`, `XK_RIGHT_AUDIT_WRITE` |
| AES-NI check | `xsec_init()` MUST detect AES-NI via CPUID; fail-fast if absent |
| Side channels | Constant-time comparisons for all cryptographic equality checks |

## `xsec.h` Public API Sketch

```c
#ifndef GENOS_XSEC_H
#define GENOS_XSEC_H

#ifndef PAL_FREESTANDING
#define PAL_FREESTANDING
#endif
#include "../../pal/include/pal.h"

typedef uint32_t xsec_key_id_t;  /* 0 = invalid */

#define XSEC_KEY_HMAC_SHA256  1U
#define XSEC_KEY_AES256_GCM   2U

/* Init — detects AES-NI, unlocks TPM, loads policy */
int          xsec_init(void);

/* Generate or import a key; returns key_id */
xsec_key_id_t xsec_key_create(uint8_t type);
void           xsec_key_destroy(xsec_key_id_t kid);

/* AES-256-GCM encrypt (in-place, appends 16B tag) */
int xsec_encrypt(xsec_key_id_t kid,
                 const uint8_t *nonce, uint8_t nonce_len,
                 uint8_t *buf, uint32_t len);

/* AES-256-GCM decrypt + authenticate */
int xsec_decrypt(xsec_key_id_t kid,
                 const uint8_t *nonce, uint8_t nonce_len,
                 uint8_t *buf, uint32_t len);

/* SHA-256 hash (result written to digest[32]) */
void xsec_sha256(const uint8_t *data, uint32_t len, uint8_t digest[32]);

/* TPM attestation quote (fills buf, returns bytes written) */
int xsec_attest_quote(uint8_t *buf, uint32_t max_len);

#endif /* GENOS_XSEC_H */
```
